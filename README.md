# EU Projekt - Docs-as-Code Template

Ovaj repozitorij služi kao početni predložak za vođenje dokumentacije EU projekata pomoću "Docs-as-Code" metodologije.

## Kako početi?
1. Klonirajte ovaj repozitorij na svoje računalo.
2. Otvorite mapu u programu **Obsidian**.
3. Administracija: Zalijepite svoj službeni sveučilišni Word dokument preko datoteke `template.docx` kako biste naslijedili sve fontove i margine.
4. Timski radite u mapi `01_Prijava_EU_Projekta/Zajednicki_Rad/`.
5. Kad ste gotovi, pritisnite tipku za slanje (Git Push). Sustav će automatski generirati gotove PDF i DOCX dokumente.

## Struktura:
- `01_Prijava_EU_Projekta` - Ovdje se nalazi glavni dokument i podijeljeni radni fajlovi za tim od 3 osobe.
- `02_Izvjestaji` - Predlošci za redovna kvartalna izvješća.
- `03_Sastanci` - Mjesto za zapisnike sa sastanaka.
- `_quarto.yml` - Konfiguracija za robota (Quarto). Ne dirati osim ako mijenjate globalne postavke.
- `references.bib` - Vaša literatura.
- `custom-header.tex` - Ako želite ubaciti logotip na PDF, otvorite ovu datoteku.
